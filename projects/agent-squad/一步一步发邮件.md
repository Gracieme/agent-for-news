# 一步一步：让 Agent 小队给你发邮件

跟着下面每一步做，做完就能收到 Agent 发的彩色邮件。

---

## 第一步：打开「终端」

1. 按键盘 **Command + 空格**， Spotlight 搜索框会出来  
2. 输入 **终端** 或 **Terminal**，回车  
3. 会打开一个黑色或白色的窗口，里面有一行字，最后是 `%` 或 `$`，这就是终端

---

## 第二步：进到 agent-squad 文件夹

在终端里**复制下面这一整行**，粘贴到终端里（Command + V），然后按**回车**：

```bash
cd /Users/geleixizhibao/Downloads/tarot-app/agent-squad
```

如果没报错，就说明已经进到正确目录了。（可以暂时不用管「报错」是什么意思，只要没有红字就行。）

---

## 第三步：复制一份「配置模板」

在终端里再复制下面这一行，粘贴，回车：

```bash
cp .env.example .env
```

这样会在 `agent-squad` 里生成一个叫 `.env` 的文件，用来放你的密钥和邮箱信息。

---

## 第四步：打开 .env 文件来编辑

有两种方式，任选一种：

**方式 A：用 Cursor / VS Code 打开**  
- 在 Cursor 左侧文件树里找到 **agent-squad** 文件夹  
- 点开，找到 **.env** 文件（可能被折叠，要展开才能看到）  
- 点一下 `.env`，在右边编辑

**方式 B：用系统自带的「文本编辑」打开**  
在终端里复制下面这一行，粘贴，回车：

```bash
open -e .env
```

会弹出「文本编辑」窗口，里面就是 `.env` 的内容。

---

## 第五步：在 .env 里填三样东西

你会看到类似这样的内容：

```
GEMINI_API_KEY=your_gemini_api_key
EMAIL_USER=jiaxinshen1208@gmail.com
EMAIL_PASS=your_app_password
```

**请把这三行改成你自己的（等号后面不要加空格）：**

1. **GEMINI_API_KEY**  
   - 把 `your_gemini_api_key` 删掉，改成你的 Gemini API Key（一串字母数字）  
   - 如果还没有：打开 https://aistudio.google.com/apikey ，登录后「Create API key」，复制过来

2. **EMAIL_USER**  
   - 已经是 `jiaxinshen1208@gmail.com` 的话就不用改  
   - 想用别的 Gmail 发信，就改成那个邮箱

3. **EMAIL_PASS**  
   - 把 `your_app_password` 删掉，改成 **Gmail 应用专用密码**（不是你的 Gmail 登录密码）  
   - 获取方法：  
     - 打开 https://myaccount.google.com/  
     - 左边点 **安全**  
     - 先开启 **两步验证**（如果还没开）  
     - 再在「两步验证」下面找到 **应用专用密码**  
     - 选「邮件」和「Windows 电脑」（或其它设备），生成  
     - 会给你 **16 个字母**，有时分成 4 组（如 abcd efgh ijkl mnop）  
     - 把这 16 个字母**连在一起**填到 `EMAIL_PASS=` 后面，**不要空格**，例如：  
       `EMAIL_PASS=abcdefghijklmnop`

改完后**保存**这个文件（Command + S），然后关掉编辑窗口。

---

## 第六步：在终端里安装依赖（只需做一次）

确保终端当前在 agent-squad 里（如果关掉了终端，再执行一次第二步的 `cd ...` 命令）。

然后复制下面这一行，粘贴到终端，回车：

```bash
npm install
```

会下载一堆依赖，等它跑完（最后出现 `%` 或 `$` 就表示好了）。

---

## 第七步：发一封测试邮件

在终端里复制下面这一行，粘贴，回车：

```bash
npm run run-once
```

脚本会跑一会儿（几十秒到一两分钟），你会看到很多输出。  
如果最后出现类似 **「已发送邮件到 jiaxinshen1208@gmail.com」**，就说明成功了。

---

## 第八步：去邮箱里查收

1. 打开你的邮箱（jiaxinshen1208@gmail.com 或你在 TO_EMAIL 里填的邮箱）  
2. 看**收件箱**，没有的话看一下**垃圾邮件**  
3. 找主题是 **「Agent 小队 · 每日推送 · 日期」** 的邮件  
4. 点开就是彩色排版的：英语习语、美妆小贴士、3 篇文献推荐

---

## 如果某一步出错了

- **提示「未配置 EMAIL_USER/EMAIL_PASS」**  
  → 回到第五步，检查 `.env` 里 `EMAIL_USER` 和 `EMAIL_PASS` 是否都填了，保存后再执行第七步。

- **提示「发送邮件失败」或「Invalid login」**  
  → 多半是 `EMAIL_PASS` 填错了。确认用的是 **应用专用密码**（16 位），不是 Gmail 登录密码；16 位要连在一起，没有空格。

- **提示「缺少 GEMINI_API_KEY」**  
  → 回到第五步，在 `.env` 里填好 `GEMINI_API_KEY` 并保存，再执行第七步。

- **找不到 .env 文件**  
  → 在 Cursor 里，`.env` 可能在列表里被折叠了，注意看 agent-squad 下面有没有「隐藏」的小文件；或用第四步的 `open -e .env` 在终端里打开。
